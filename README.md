## How to Run the App

### Backend
1. `cd BACKEND`
2. `npm install`
3. `node server.mjs`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`

### Usage
1. Register a new user
2. Login with your credentials
3. Create posts with images
4. View posts on the main page

### .Env
1. Remember to add your own .env with the mongo connection string in backend
